# Scratch_Pong-with-AI

Versión de Pong para jugar 1 vs 1 en el mismo dispositivo o contra la máquina.

## Controles:
### Multijugador:
#### -Jugador 1:
  
  w: Arriba.
  
  s: Abajo.
  
#### -Jugador 2:
  
  u: Arriba.
  
  j: Abajo.
  
  **Click con el ratón sobre el jugador 2: Activar o desactivar la IA.**
  
  d: Ajustar difilcutad de la IA (1-20).
