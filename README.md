# Scratch_Pong-with-AI
A version of the game Pong to play 1 vs 1 in same device or against the device.

## Controls:
### Multiplayer:
#### -Player 1:
  w: up.
  
  s: down.
#### -Player 2:
  u: up.
  
  j: down.
  
  **click on player 2: set on/off AI mode.**
  
  d: set difficulty (1-20).
